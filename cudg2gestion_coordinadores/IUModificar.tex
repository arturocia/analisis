\subsection{Pantalla: Modificar Coordinador}

\subsubsection{Objetivo}
Permite al usuario modificar la información correspondiente a un Coordinador seleccionado.

\IUfig[0.6]{CUDG2/ModificarCoordinador.png}{IUModificarCoordinador}{Modificar Coordinador.}

\subsubsection{Entradas}
Datos editables de Coordinador \ref{dd:DatosEditablesCoordinador}.

\subsubsection{Salidas}
\begin{itemize}
 \item Se muestran los datos editables actuales del Coordinador que se desea modificar.
 \item Se muestra el mensaje (MSG-4) \ref{MSG4} 
\end{itemize}

\subsubsection{Comandos}
\begin{itemize}
 \item \IUbutton{Aceptar} Modifica el registro siempre y cuando los datos sean correctos.
 \item \IUbutton{Cancelar} Cancela la operación. Lleva a la pantalla \IUref{IUGestionCoordinadores}{Gestion de Coordinadores}.

\end{itemize}
